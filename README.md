# 📄 CV Web — Luis Orellana

Currículum vitae en formato web creado con **HTML5** y **Bootstrap 5**.

## 🎯 Descripción
Este proyecto es parte del Bootcamp Full Stack. Se creó un CV web que incluye:
- Datos de contacto
- Perfil profesional
- Experiencia y educación
- Habilidades técnicas con barras de progreso
- Proyectos destacados
- Diseño responsive

## 🛠️ Tecnologías usadas
- HTML5 semántico
- Bootstrap 5
- Bootstrap Icons
- Google Fonts (Inter)

## 📸 Vista previa
Abre el archivo `index.html` en tu navegador para verlo.
